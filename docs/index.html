Features

Everything you need for production AI inference

Drop-in OpenAI replacement that routes requests across multiple backends with enterprise-grade reliability.

🔀
Multi-Backend Routing
Groq, OpenRouter, Gemini, Ollama, vLLM, SGLang — all behind a single OpenAI-compatible API.
🧪
A/B Testing
Weighted traffic splitting with sticky sessions. Compare model performance in production.
⚡
Circuit Breaker
Automatic failure detection with half-open recovery. Prevents cascading failures across backends.
🔒
Multi-Tenant Auth
API key authentication with per-tenant rate limits using token bucket algorithm.
📡
SSE Streaming
Full Server-Sent Events streaming support. Compatible with OpenAI client libraries.
📊
Prometheus + Grafana
Request rates, latencies, token usage, and backend health — all auto-instrumented.
